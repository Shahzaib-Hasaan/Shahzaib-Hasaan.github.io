Supervised and Unsupervised Learning

Posted on October 20, 2023 by Shahzaib Hasaan

"Machine Learning is a type of Artificial Intelligence that allows computers to learn without being explicitly programmed" -Arthur Sameul

Machine Learning models are trained on data, and they learn the mapping or patterns in data so that they can make predictions based on their learning.There are two main types of machine learning:

  1. Supervised Learning
  2. Unsupervised Learning

In this post we are going to discuss the difference between supervised and unsupervised learning. But before that let's discuss what is supervised and unsupervised learning.

Supervised Learning

In supervised learning, the model is trained on labeled data.Labeled Data means that the data contains the features(the things we know to predict something) as well as labels(which we want to predict). Let's suppose we want to predict the price of houses through supervised learning , we need labeled data which conatins important features like:

  • area of the house(5 marla, 10 marla etc)
  • number of rooms(2,3,5 etc)
  • number of bathrooms(1,2,3 etc)
  • location of the house(city or village)
  • condition of the house(new or old)

And the label will be the price of the house. So our dataset will be labeled which means it will contain data of different houses, each row will contain the features of a house and the price of that house.Below is a sample dataset given for this putpose:

Area Rooms Bathrooms Location Condition Price
5 marla 2 1 City Old 1000000
10 marla 3 2 Village New 2000000
5 marla 2 1 City New 1500000
10 marla 3 2 Village Old 1800000

We can see that each row(group of features) has a label(price) associated with it. So this is a labeled dataset and we can use it to train our model.

Unsupervised Learning

It is a type of machine learning in which the model is trained on unlabeled data.Unlabeled Data means that the data only contains features and no information about what label is associated with each row. Let's suppose we want to cluster the houses into different groups based on their features, we can use unsupervised learning for this purpose. We will use the same dataset as above but this time we will not include the price column. So our dataset will look like this:

Area Rooms Bathrooms Location Condition
5 marla 2 1 City Old
10 marla 3 2 Village New
5 marla 2 1 City New
10 marla 3 2 Village Old

We can see that there is no price column in this dataset. So this is an unlabeled dataset and we can use it to train our model.

Ok so now we know what is supervised and unsupervised learning. Let's discuss the difference between them.

Difference between Supervised and Unsupervised Learning

The main difference between supervised and unsupervised learning is that in supervised learning the model is trained on labeled data while in unsupervised learning the model is trained on unlabeled data.
But if you thought deeply about these 2 types of learning there are some other differences as well.

Supervised Learning Unsupervised Learning
It is a type of machine learning in which the model is trained on labeled data. It is a type of machine learning in which the model is trained on unlabeled data.
In this type of learning, the model learns the mapping between features and labels. In this type of learning, the model learns the patterns in data.
It is used for classification and regression problems. It is used for clustering and association problems.
